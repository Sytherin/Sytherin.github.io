# Portfolio Website - Kent Cajes

A modern, responsive portfolio website showcasing my work as a Bachelor of Science in Information Systems (BSIS) student.

## Features

- **Home Page**: Hero section with introduction and skills preview
- **About Page**: Detailed information about my background, education, and skills
- **Contact Info Page**: Contact details and a contact form
- **Photos Page**: Gallery for showcasing photos and projects

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox, and animations)
- JavaScript (vanilla JS for interactivity)
- Responsive design for all devices

## Skills Highlighted

- Vue.js
- Laravel
- C++
- Python
- MySQL
- HTML & CSS

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/Sytherin/portfolio.git
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser
